* Описание
Данный продукт разработан в учебных целях. Десктопное приложение для автоматизации работы администратора ресторана позволяет управлять бронированием столиков, отслеживать свободные места, назначать время визита гостей и вести историю заказов. Приложение реализовано на C# с использованием Windows Forms, поддерживает валидацию введенных данных и сохраняет информацию о бронированиях в файлы.

* Установка
Для установки программы необходимо перейти в раздел releases, скачать актуальную версию файла RestaurantBooking.exe из последнего релиза, после чего открыть папку загрузок в файловом менеджере и запустить приложение двойным щелчком мыши.

* Использование
Данное десктопное приложение предназначено для администраторов и хостес ресторана. При первом запуске открывается главное окно с сеткой столиков. Пользователь может создать новую бронь, указав имя гостя, контактный номер, количество персон, дату и время. Приложение визуально отображает занятые и свободные столики, контролирует вместимость и предотвращает двойное бронирование на одно и то же время. Все данные хранятся локально на компьютере в структурированном текстовом формате.

* Разработка
Проект разрабатывается в рамках учебной практики индивидуально и не предполагает привлечения сторонних участников. Десктопное приложение создается на языке C# с использованием Windows Forms с целью демонстрации навыков объектно-ориентированного программирования, работы с элементами управления WinForms и реализации логики бронирования.
